# Telecom Churn Prediction

A machine learning project for predicting telecom customer churn using exploratory data analysis, preprocessing, feature engineering, and classification techniques.

## Overview

Customer churn prediction is an important business problem for telecom companies. This project analyzes customer behavior and builds a predictive model to identify users who are likely to leave the service.

## Workflow

- Business problem understanding
- Data cleaning and preprocessing
- Exploratory data analysis
- Feature engineering
- Model building
- Model evaluation
- Key business insights

## Tech Stack

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
